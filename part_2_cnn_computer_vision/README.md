\#                TASK 6: CNN CONCEPT EXPLANATION

\# ============================================================

1. What is Convolution?



Convolution is a process where filters scan the image

to detect important features like edges, textures,

and patterns.



\---------------------------------------------------



2\. Why is Pooling Used?



Pooling reduces the size of feature maps.

This helps reduce computation and prevents overfitting.



\---------------------------------------------------



3\. Why is ReLU Commonly Used?



ReLU introduces non-linearity into the neural network.

It helps the model learn complex patterns efficiently.



Formula:

ReLU(x) = max(0, x)



\---------------------------------------------------



4\. Why are CNNs Better than Regular Networks?



CNNs automatically learn image features using filters.



Advantages:

\- Better feature extraction

\- Fewer parameters

\- Faster training

\- Better image understanding
